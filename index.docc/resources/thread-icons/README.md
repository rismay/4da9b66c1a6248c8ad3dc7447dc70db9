Thread icon SVG templates.

Naming:
- thread-icon-p{0..3}-s{0|1|x}.svg

Where:
- pN = priority major bucket
- s0 = severity S0 (red)
- s1 = severity S1 (orange)
- sx = other/unspecified severity (gray)

DocC references images by basename (no extension), e.g.
@PageImage(purpose: icon, source: "thread-icon-p1-s0", alt: "P1 S0")
